## üìÑ FAQs

### What is a data breach?

A data breach occurs when sensitive or confidential information is accessed without authorization. This can include passwords, email addresses, and other personal data.

### How do I know if my email was compromised?

The hibp-lookup application scans known databases of breached data. If it finds your email in any of these records, it will notify you.

### Is it safe to use hibp-lookup?

Yes. The application does not collect or store your data. It only checks against known breaches.
